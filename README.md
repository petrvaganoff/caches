# Caches

Библиотека классов, содержит реализацию кэшей на С++:

- Обобщённая (template) реализация LRU-кэша (Least Recently Used Cache) - структура данных, которая хранит ограниченное количество элементов и автоматически удаляет наименее недавно использованные. При доступе к элементу он становится "самым свежим" (первым в списке). При переполнении удаляется "самый старый" (последний в списке).
